ZWIN LOGO KIT
=============

Files (all PNG, 4x resolution; lockups and mark have transparent backgrounds):
- zwin-lockup-horizontal-navy.png  — primary logo, for light backgrounds
- zwin-lockup-horizontal-white.png — for dark backgrounds
- zwin-lockup-stacked-navy.png     — stacked version, light backgrounds
- zwin-lockup-stacked-white.png    — stacked version, dark backgrounds
- zwin-mark.png                    — Z symbol only, transparent
- zwin-avatar-cream.png            — square avatar / app icon tile (cream)
- zwin-avatar-navy.png             — square avatar / app icon tile (navy)
- zwin-wordmark-navy.png           — wordmark only

Brand basics
------------
Wordmark font: Archivo Bold (Google Fonts), all caps, letter-spacing ~0.28em
Navy text:     #17418f
Dark surface:  #0b1626
Avatar cream:  #f5f3ee
Mark:          faceted Z, navy-to-cyan gradient

Usage
-----
- Use the navy lockups on white/light backgrounds, white lockups on dark.
- Use the mark alone at small sizes (favicons, UI corners).
- Use the square tiles for social avatars and app icons.
